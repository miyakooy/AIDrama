# TensorsLab Studio 魔法向导模式 (Magic Creator) 产品设计文档 (PRD)

## 1. 概述
**TensorsLab Studio** 引入了全新的「魔法向导模式 (Magic Creator)」，旨在大幅降低 AI 影视创作的门槛。该模式将原有的离散式工作台（剧本、角色、场景、分镜、渲染）整合成了一个类似搜索引擎的极简单入口，实现“一句话生成大片”的 OiiOii 风格全自动管线体验，同时完美兼容专业创作者的手动微操需求。

---

## 2. 核心功能与交互流程

### 2.1 极简首页 (Hero Section)
- **视觉风格**：深色模式 (Dark Mode)，大面积毛玻璃质感与赛博朋克色调的氛围光效，背景带噪点遮罩。
- **导航栏**：
  - 左侧：产品 Logo + `Magic Creator` 标题。
  - 右侧：提供 `专业工作台` 入口，方便高级用户随时切换至传统的分模块仪表盘。
- **快捷动作按钮 (Action Chips)**：
  - 提供一组横向排列的功能快捷入口，如 `Seedance 2.0故事动画`、`快速生图/视频`、`音乐概念短片` 等，引导用户的创作方向。

### 2.2 智能输入框 (Omni-Input)
- 居中放置的大尺寸文本框。
- **占位符引导**：“拖拽/粘贴 图片到这里，来试试【角色】、【风格】参考” / “例如：一个赛博朋克风格的刺客...”。
- **附属功能**：
  - **✨ 帮我润色**：通过 LLM 优化简短提示词。
  - **🎲 随机灵感**：自动填入系统预设的优质 Prompt。
- **一键生成按钮**：点击后触发全自动创作管线，按钮进入 Loading 状态。

### 2.3 全自动生成管线 (Auto-Pipeline)
点击生成后，系统接管操作并按以下顺序自动在后台调度：
1. **LLM 剧本解析**：调用配置的文本大模型（如 DeepSeek），将简短梗概扩写为标准的分镜脚本格式。
2. **工作区初始化**：静默创建新项目，自动生成 UUID 并切换至该项目的数据上下文。
3. **资产提取与构建**：调用 `importFullScript` 服务，利用正则或大模型自动切分场景、提取出场角色、生成角色小传与第一集初始镜头列表。
4. **无缝流转**：数据准备完毕后，自动跳转至 `专业导演工作台 (Director Panel)`，此时剧本分镜已全部加载就绪。

### 2.4 OiiOii 风格沉浸式进度遮罩 (Progress Overlay)
为了缓解长时间 API 调用的等待焦虑，设计了全屏沉浸式的加载过场动画：
- **动画核心**：采用立体透视的菱形/方块复合自转动画，内嵌渐变发光核心，展现“AI 构建世界”的科技感。
- **状态流转面板**：
  - 主标题：“AI 引擎正在构建你的世界”。
  - 进度信息卡：包含动态扫描光条，实时展示底层管线的当前执行步骤（如：“初始化 AI 引擎...” -> “构思剧本与分镜...” -> “构建底层资产...” -> “进入导演台...”）。
  - 底部提示：“大模型调度中，请勿关闭窗口...”。

## 3. 架构设计与技术实现

### 3.1 前端组件
- **路径**: `src/components/panels/magic-creator/MagicCreator.tsx`
- **状态管理**: 结合 React `useState` (管理 Prompt 与局部生成状态) 与 Zustand `useProjectStore`, `useMediaPanelStore` (管理项目创建与路由切换)。

### 3.2 演示模式 (Demo Mode) 支持
针对汇报与演示场景，当前实现了 **Demo Mock 逻辑**：
- 点击生成后，**不**实际调用耗时的 LLM 扩写与资产构建。
- 而是利用 `setTimeout` 模拟各阶段的加载状态，随后直接调用 `switchProject(DEMO_PROJECT_ID)` 加载内置的高质量演示项目（“灌篮少女”），并跳转至导演台。

### 3.3 扩展与未来规划
- **多模态输入**：未来需真正对接图片拖拽识别，自动将图片转化为 `Image Prompt` 或传入角色库作为参考图。
- **全流程渲染打通**：在跳转导演台后，可进一步触发 `handleGenerateAllImages` 和 `generateAllGroups`，实现“输入一句话，最后直接出视频文件”的极致终态。
